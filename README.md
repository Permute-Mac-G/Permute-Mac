## 💡 Overview

<a href="#permute">Permute</a> serves Mac users who encounter format compatibility problems regularly — the video that won't play in a specific application, the audio format that a device doesn't support, the image format that a service won't accept — and who want a conversion solution that works without requiring technical knowledge.

The <a href="#permute">home user conversion use case</a> covers the common everyday conversions — converting iPhone videos from HEVC to MP4 for sharing with Windows users, converting downloaded audio to a format the car stereo accepts, converting RAW photos to JPEG for web upload, or converting any file to the format a specific context requires. Permute handles these without settings adjustment for most cases. The <a href="#permute">content creator workflow</a> uses Permute for preparing media in specific platform-required formats — converting recorded video to the codec and container a video editor expects, preparing audio in the format a podcast host requires, or converting images to the specification a web platform mandates.

The <a href="#permute">professional and semi-professional use case</a> benefits from batch processing for converting collections of files — a photographer converting a session of RAW files to JPEG, a podcaster converting a library of audio files to a consistent format, or anyone with a large collection requiring format standardization. <a href="#permute">No subscription required</a> for the direct purchase version — Permute is a one-time purchase from the Mac App Store in addition to its Setapp availability.

<a href="#permute">Clean minimal interface</a> reflects Permute's design focus — presenting conversion as a simple task in a simple interface rather than wrapping it in professional complexity that casual users find intimidating. The interface design is one of Permute's most cited qualities by users who have tried competing converters. <a href="#permute">Regular format support updates</a> keep Permute compatible with new formats as they emerge — WebP image support, HEVC video, and other format additions respond to the formats that Mac users encounter in current use.

<a href="#permute">Conversion queue management</a> shows active and completed conversions in a queue — monitoring progress, identifying completed files, and managing batch jobs from a single interface view. <a href="#permute">Apple Silicon optimization</a> uses the M-series chip's dedicated media engine for encoding tasks — delivering faster conversion on Apple Silicon Macs than Intel-based software encoding, making large file processing significantly more practical on newer hardware.

## 🙋 FAQ

<details>
<summary>How does Permute compare to HandBrake?</summary>

HandBrake provides more granular encoding control for professional video encoding scenarios. Permute offers a simpler interface with broader format coverage for everyday conversion needs. Both are capable; the choice depends on whether encoding control or simplicity matters more.

</details>

<details>
<summary>Does Permute support HEVC video conversion?</summary>

Yes. HEVC (H.265) is supported for both input and output conversion.

</details>

<details>
<summary>Can Permute convert image files?</summary>

Yes. Image conversion covers JPEG, PNG, HEIF, WebP, TIFF, and other image formats.

</details>

<details>
<summary>Does Permute convert PDF files?</summary>

Yes. PDF conversion is supported. Check current Permute documentation for specific PDF conversion capabilities.

</details>

<details>
<summary>Does Permute maintain quality through conversion?</summary>

Permute uses quality-optimized default settings for each format. Lossy format conversion involves some quality reduction by nature; lossless formats maintain full quality. Settings are adjustable for users who need specific quality levels.

</details>

Check the Mac App Store listing for current minimum macOS requirements.

</details>
